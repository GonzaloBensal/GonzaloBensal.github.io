## Description

    First version of a digit counter, made for practice purposes only ✌

## Observations
    Max length supported: 308

    Only accepts integer numbers, not float

## Examples
    1. 345 -> 3 digits
    2. -1001 -> 4 digits
    3. 0 -> 1 digit
    4. null -> 0 digits
    5. a number with more than 308 digits -> 0 digits (FATAL! don't do this 😂)
